Fix 'File/Application is damaged and cannot be opened'

2023/04/24 ・ system

#solution #macos

You may get errors “File/App is damaged and cannot be opened. You should move it to Trash.” when opening files or applications on macOS Ventura/Monterey/Big Sur/Catalina, especially on M1 or M2 Macs. This article will tell you the solution.

Remove the extended attributes of the damaged file or app

  1. Open the Terminal.
  2. Type in the following command and hit Enter.
sudo xattr -d com.apple.quarantine file_path

You need to delete file_path, the drag and drop the file or application to Terminal and hit Enter.

This is an example:

Ma1icious@host ~ % sudo xattr -r -d com.apple.quarantine /Applications/Clash\ for\ Windows.app
